RetroStore

WiFi Configuration

The WiFi credentials have been uploaded. The RetroStore card will now reboot and attempt to connect to the specified network. If the connection is successful, the LED will light green for 3 seconds. If the RetroStore card could not connect, the LED will turn red. The RetroStore card will store the credentials. In order to reset the WiFi credentials, hold the push button while turning on the RetroStore card.
